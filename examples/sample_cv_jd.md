# Ví dụ Mẫu CV và JD (Sample Input Case)

File này cung cấp ví dụ mẫu về CV và Job Description (JD) để chạy thử nghiệm AI Agent Reviewer.

---

## 1. Mẫu CV Ứng viên (Sample CV)

**Nguyễn Văn A**
Data Analyst | Email: nguyenvana@email.com | Phone: 0901234567

**Tóm tắt bản thân:**
Data Analyst nhiệt huyết với 2 năm kinh nghiệm làm việc với dữ liệu. Thành thạo Python, SQL và Power BI. Luôn sẵn sàng học hỏi công nghệ mới và đóng góp cho doanh nghiệp.

**Kinh nghiệm làm việc:**

**Công ty Cổ phần Công nghệ XYZ (01/2024 - Hiện tại)**
*Vị trí: Data Analyst*
- Viết câu lệnh SQL để truy xuất dữ liệu từ cơ sở dữ liệu PostgreSQL.
- Làm sạch và xử lý dữ liệu thô bằng ngôn ngữ Python.
- Tạo báo cáo và dashboard bằng Power BI cho ban giám đốc.
- Tham gia hỗ trợ các phòng ban khác khi có yêu cầu về dữ liệu.

**Công ty TNHH Giải pháp ABC (06/2022 - 12/2023)**
*Vị trí: Junior Data Assistant*
- Hỗ trợ nhập dữ liệu và kiểm tra tính chính xác của dữ liệu.
- Viết báo cáo tuần bằng Excel.

**Kỹ năng:**
- Ngôn ngữ: Python, SQL, R
- Cơ sở dữ liệu: PostgreSQL, MySQL
- Công cụ BI: Power BI, Excel
- Kỹ năng mềm: Giao tiếp, Làm việc nhóm, Quản lý thời gian

## 2. Mẫu Mô tả Công việc (Sample JD)

**Vị trí: Senior Data Analyst / Analytics Engineer**
*Địa điểm: Hà Nội | Mức lương: Thỏa thuận*

**Mô tả công việc:**
- Xây dựng và duy trì hệ thống dữ liệu, ETL pipeline từ nhiều nguồn khác nhau về Data Warehouse.
- Viết các truy vấn SQL phức tạp và tối ưu hóa hiệu năng truy vấn trên PostgreSQL / BigQuery.
- Thiết kế dashboard và báo cáo trực quan trên Power BI / Tableau phục vụ Ban giám đốc và các Trưởng phòng.
- Phân tích hành vi người dùng, phối hợp với đội ngũ Product thực hiện A/B Testing.
- Tự động hóa các báo cáo định kỳ bằng Python.

**Yêu cầu công việc:**
- Tối thiểu 2 năm kinh nghiệm ở vị trí tương đương.
- Thành thạo SQL (viết truy vấn nâng cao, window functions, tối ưu hóa query).
- Sử dụng thành thạo Python (Pandas, NumPy) cho xử lý và tự động hóa dữ liệu.
- Kinh nghiệm làm việc với BI Tools (Power BI / Tableau).
- Có tư duy sản phẩm, khả năng phân tích chỉ số kinh doanh tốt.
- Ưu tiên ứng viên có kinh nghiệm với Docker, Git, AWS hoặc Google Cloud Platform.
